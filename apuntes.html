Apuntes HTML

Estos son algunos apuntes sobre HTML.

Elementos Básicos

  • <html>: Define el documento HTML.
  • <head>: Contiene metadatos y enlaces a recursos.
  • <body>: Contiene el contenido visible de la página.
  • <h1,h2,h3,h4,h5,h6>: Representa distintos tipos de headers
  • <p>: Representa un párrafo

Otros elementos

  • <a>: Define un enlace: <a target="_blank" href="https://www.youtube.com/watch?v=odDOEzSLcJs">enlace </a>
  • <span>: Crea una sección de texto destacado: <span style= "color:red">
  • <img>: Inserta una imagen. Se puede cerrar en la misma etiqueta (<img />)
  • <ul>: Crea una lista desordenada.
  • <ol>: Crea una lista ordenada.
  • <table>: Crea una tabla.
